# 点焊工程与算法统一认知文档（归档版）

# 第一章 预备知识

## 一、核心术语统一说明（非常重要）

在本文中，统一使用以下三个术语：

- **融合（Fusion）**：物理过程
- **融合点（Fusion Point）**：几何 / 能量作用位置
- **焊核（Weld Nugget）**：融合完成后的实体结果

> ❗ 后续所有讨论，均基于这三者的严格区分

------

## 二、什么是「融合」

### 2.1 融合的工程定义

**融合（Fusion）**是指：

> **在焊接过程中，母材（及可能的填充材料）在高温作用下发生熔化，并在液态条件下相互扩散、混合，最终在冷却后形成连续冶金结构的物理过程。**

关键点：

- 必须 **发生熔化**
- 必须 **发生冶金结合**
- 冷却后形成 **连续金属晶体结构**

📌 只要缺少其中任意一步，都不能称为“成功融合”。

------

### 2.2融合不是“结果”，而是“过程”

这一点非常关键：

- ❌ “已经焊住了” → 这是结果判断
- ✅ “是否发生了充分融合” → 这是过程判断

工程与算法关注的永远是：

> **融合是否按预期发生**

------

### 2.3 融合 ≠ 单纯熔化

> **熔化（Melting）是状态变化，融合（Fusion）是冶金行为。**

| 项目             | 熔化          | 融合 |
| ---------------- | ------------- | ---- |
| 是否变液态       | ✅             | ✅    |
| 是否发生元素扩散 | ❌（可能没有） | ✅    |
| 是否形成冶金结合 | ❌             | ✅    |
| 是否形成有效焊核 | ❌             | ✅    |

📌 工程上存在的典型误判：

- “表面已经化了” ❌
- “看起来粘住了” ❌

**只有发生融合，焊核才成立。**

## 三、什么是「融合点」

### 3.1 融合点的定义

**融合点（Fusion Point）**指的是：

> **焊接过程中，能量（电流 / 电弧 / 激光）实际作用到工件，并引发材料开始熔化的几何位置。**

它是一个：

- **位置概念**
- **能量输入中心**
- **控制与规划对象**

而不是一个实体。

------

### 3.2 不同焊接方式下的融合点

| 焊接方式 | 融合点含义                        |
| -------- | --------------------------------- |
| 电弧焊   | 电弧中心在工件表面的投影点        |
| 激光焊   | 激光焦点在材料上的落点            |
| **点焊** | 电极压力 + 电流集中作用的接触中心 |

📌 特别强调（点焊）：

> 融合点 ≠ 两个零件的几何交点
> 而是 **热量真正进入材料的位置**

------

### 3.3 融合点不是“大小”，而是“位置”

因此以下说法是**错误的**：

- ❌ “融合点直径是多少”
- ❌ “这个融合点太小了”

正确表达应该是：

- ✅ “焊核直径偏小”
- ✅ “融合不足导致焊核不完整”

------

## 3.4 融合点 ≠ 最高温点（很多人会误会）

> **融合点是“控制与规划上的能量作用中心”，
>  并不严格等同于瞬时最高温点。**

原因：

- 点焊中存在：
  - 接触电阻不均
  - 材料热扩散
  - 电极冷却

📌 算法上你“锁定融合点”是对的
 📌 物理上温度峰值可能略有偏移

## 四、什么是「焊核」

### 4.1 焊核的定义

**焊核（Weld Nugget）**是指：

> **在融合过程完成后，熔化区域冷却凝固所形成的、具有连续冶金结合的实体金属区域。**

关键属性：

- 是 **实体**
- 有 **体积**
- 有 **直径、形状、强度**

------

### 4.2 焊核只在“焊接完成后”存在

从时间维度看：

#### 焊接前

- 只有融合点
- 没有焊核

#### 焊接中

- 正在发生融合
- 焊核尚未稳定

#### 焊接后

- 熔池凝固
- 焊核形成并可检测

------

### 4.3 融合点与焊核的关系

```
融合点（位置）
   ↓
融合过程（冶金 + 熔化）
   ↓
焊核（实体）
```

工程上常说：

```
融合点 ≈ 焊核几何中心
```

但注意：

- 一个是 **点**
- 一个是 **体**
- 概念不能混用

------

好，这一问**正中“最后一块拼图”**。
你前面已经把 **融合 / 融合点 / 焊核** 全部吃透了，现在问 **“焊缝是啥”**，其实是在做一次**概念边界收口**。

我按你前面整理文档的风格来，**不简化、不偷懒、可直接归档**。

------

## 五、什么是「焊缝」

焊缝是什么（Weld Seam / Weld Bead）

> **焊缝，是沿焊接路径形成的、由焊接金属凝固后构成的“连续几何实体”；
> 它是“线性焊接”的结果，而不是焊接的物理本质。**

换句话说：

- **融合** → 物理过程
- **焊核** → 点焊结果
- **焊缝** → 连续焊的“几何外显形态”

------

### 5.1 焊缝的标准定义（不掺点焊）

**焊缝（Weld Seam / Weld Bead）**是指：

> **在连续焊接过程中，随着热源沿焊接路径移动，熔池不断形成并凝固，最终在工件表面和内部形成的一条连续冶金结合区域。**

关键词拆解：

- **连续焊接**
- **沿路径移动**
- **连续形成**
- **有长度、有截面**

------

### 5.2 焊缝 vs 焊核（你现在最需要的对照）

| 对比项             | 焊核（点焊） | 焊缝（连续焊） |
| ------------------ | ------------ | -------------- |
| 维度               | 点 / 体      | 线 / 体        |
| 是否连续           | ❌ 离散       | ✅ 连续         |
| 是否有路径         | ❌            | ✅              |
| 形成方式           | 单次通电     | 热源连续移动   |
| 是否强调外形       | 较弱         | **非常强调**   |
| 是否存在“焊缝高度” | ❌            | ✅              |

一句话总结：

> **焊核是“一个点焊成功了”，
> 焊缝是“无数次融合沿路径叠加的结果”。**

------

### 5.3 从时间 + 空间维度理解焊缝（非常重要）

 点焊（焊核）

```
时间：一次
空间：固定点
结果：一个焊核
```

------

连续焊（焊缝）

```
时间：连续
空间：沿路径 P(s)
结果：焊核在路径上的“连续并集”
```

你可以把焊缝理解成：

> **焊核沿路径的积分结果**

数学直觉版：

```
焊缝 ≈ ∫ 焊核(s) ds
```

------

### 5.4 焊缝“到底是表面的，还是内部的？”

这是一个**非常常见但容易混淆的问题**。

正确答案是：**两者都是**

#### **外部焊缝（你看到的）**

- 焊缝高度
- 焊缝宽度
- 焊道外观

👉 **这是几何形态**

------

#### 内部焊缝（真正决定强度的）

- 熔深
- 熔合区
- 冶金结合区

👉 **这是力学与冶金本体**

📌 这也是为什么：

> **焊缝“好不好看” ≠ 焊接“牢不牢”**

------

### 5.5 焊缝的几个关键几何量（你后面一定会用到）

#### 焊缝宽度（Width）

- 表面展开宽度
- 与热输入、摆动有关

------

#### 焊缝高度（Reinforcement）

- 高于母材表面的高度
- 主要来自填充金属

📌 **高度大不代表强度大**

------

#### 熔深（Penetration）

- 融合进入母材的深度
- 决定焊接是否有效

👉 **熔深才是焊缝强度的核心指标**

------

### 5.6 焊缝与“虚焊”的关系（重要）

你前面已经理解虚焊，现在可以统一解释了。

#### 正常焊缝

```
焊缝金属
█████████
母材熔合区
█████████
```

------

#### 虚焊焊缝（非常危险）

```
焊缝金属
█████████   ← 看起来很漂亮
─────────   ← 未熔合界面
母材
```

📌 所以：

> **虚焊不是“没有焊缝”，
> 而是“有焊缝，但没有有效融合”。**

------

### 5.7 为什么点焊里几乎不说“焊缝”

这是你这个阶段**非常到位的问题**。

原因很简单：

- 点焊 **没有路径**
- 没有连续熔池
- 没有线性几何体

所以：

> **点焊只有焊核，没有焊缝**

如果有人在点焊里说“焊缝”——
👉 **90% 是口语误用**

------

### 5.8从算法 / 软件视角看焊缝（你一定会喜欢）

#### 连续焊接里，算法关心什么？

- 路径曲线 P(s)
- 每个 s 点的：
  - 融合点
  - 法线
  - 姿态
  - 热输入

------

#### 算法眼中的焊缝

> **焊缝不是一个物体，而是：**
>
> - 一条路径
> - 一组局部坐标系
> - 一套随路径变化的工艺参数

------

#### 点焊 vs 连续焊（算法差异）

| 项目 | 点焊   | 连续焊   |
| ---- | ------ | -------- |
| 路径 | ❌      | ✅        |
| 法线 | 离散   | 连续     |
| 姿态 | 点到点 | 连续插值 |
| 焊核 | 单个   | 串联     |
| 焊缝 | ❌      | ✅        |

------

## 概念“终极关系图”

```
融合点（位置）
   ↓
融合（物理过程）
   ↓
焊核（点焊结果）
   ↓（沿路径连续）
焊缝（连续焊结果）
```

------

一句话终极总结

> **焊缝是连续焊接中，由融合过程沿路径连续发生而形成的冶金结合实体；
> 它是焊核在空间路径上的连续叠加结果，而不是焊接的物理起点。**

# 第二章：为什么点焊要求姿态“垂直于融合点法线”

------

## 五、点焊的物理目标是什么？

一句话总结：

> **点焊的目标不是“通电”，而是“在受控条件下形成稳定焊核”。**

而稳定焊核的前提是：

- 能量分布均匀
- 熔化区域可控
- 冷却过程对称

------

## 六、垂直姿态的真正意义

### 6.1 核心结论（非常重要）

> **垂直不是目的，均匀分布才是目的。**

- 垂直只是最简单、最稳定的实现方式
- 它能天然保证：
  - 能量轴对称
  - 熔池对称
  - 冷却对称
- 压紧阶段局部形变不可避免，但垂直姿态仍是能量均匀分布的最佳近似方式；微小形变会引起融合点偏移，但对整体焊核稳定性影响有限。

------

### 6.2 非垂直会发生什么？

当姿态偏离融合点法线：

- 能量沿切向扩散
- 熔池向一侧偏移
- 熔深不均
- 冷却收缩偏心

结果可能是：

- 焊核偏小
- 焊核形状异常
- 虚焊 / 假焊

------

## 七、从“分布不均匀”的角度统一解释一切

你提出的这个判断是**本质性结论**：

> **真正的问题不是“垂直 or 不垂直”，
> 而是“能量、温度、金属流动是否分布均匀”。**

垂直姿态的作用只是：

- 最大化分布均匀性
- 最大化工艺容错性
- 最小化参数耦合复杂度

# 第三章：点焊中的几何与算法模型

------

## 八、融合点法线如何从 CAD / 曲面计算

### 8.1 CAD 中的几何基础

点焊在 CAD 中对应的是：

- 面与面接触
- 曲面与曲面接触
- 在接触区域选定融合点

------

### 8.2 平面法线

若融合点位于平面：

```
Ax + By + Cz + D = 0
n = (A, B, C)
```

------

### 8.3 曲面法线（更常见）

参数曲面：

```
S(u,v)
```

法线计算：

```
Su = ∂S/∂u
Sv = ∂S/∂v
n = normalize(Su × Sv)
```

------

### 8.4 点焊的“双面问题”

点焊涉及两个零件：

常见工程做法：

```
n = normalize(n1 + n2)
```

含义是：

> **取等效压紧方向作为融合点法线**

------

## 九、为什么焊接算法本质是“法线跟踪”

### 9.1 焊接控制的真正对象

机器人并不是在控制“点”，而是在控制：

> **TCP 位姿（位置 + 姿态）**

而姿态最稳定的定义方式就是：

- 用融合点法线定义一个轴

------

### 9.2 融合点局部坐标系

```
Z = 融合点法线
X = 任意稳定切向
Y = Z × X
```

点焊特点：

- Z 轴必须严格对齐
- 绕 Z 轴旋转是自由的

------

### 9.3 法线跟踪的数学表达

```
z_tcp · n ≥ cos(θ_max)
```

这就是现场说的：

> “焊枪姿态误差 ≤ ±θ°”

------

## 十、点焊的几何约束模型

### 10.1 位置约束

```
TCP_pos = 融合点 + d · n
```

- `d`：电极伸出 / 压紧余量

------

### 10.2 姿态约束

```
R · Z = n
```

------

### 10.3 自由度说明

- 点焊只有 **5 个有效约束**
- 绕法线旋转自由

# 第四部分：焊核、虚焊与能量分配

------

## 十一、什么是虚焊 / 假焊

### 11.1 定义

> **虚焊是指焊接过程中母材未充分融合，焊核未形成或极小，仅靠表面附着维持连接的失效状态。**

------

### 11.2 本质原因

```
融合不足 → 熔深不足 → 焊核不连续
```

而不是：

- 焊缝不好看
- 表面高度不够

------

### 11.3 为什么外观常常“看不出来”

- 焊缝高度 ≠ 熔深
- 表面金属可能已熔化
- 母材内部却未融合

------

# 第五部分：压力与电流的耦合关系（点焊核心）

------

## 十二、点焊的能量模型

```
Q = I² · R · t
```

- 热量来自电阻
- 压力通过改变 R 参与焊接

------

## 十三、压力的真实作用

压力的作用不是“压住”，而是：

- 控制接触面积
- 控制接触电阻
- 稳定电流路径

```
压力 ↑ → 接触面积 ↑ → R ↓  
但压紧会产生局部弹塑性形变 → 接触电阻分布可能轻微偏移
```

------

## 十四、四种典型组合（完整归档）

| 压力 | 电流 | 结果            | 工程评价 |
| ---- | ---- | --------------- | -------- |
| 合适 | 合适 | 焊核稳定        | ✅        |
| 大   | 小   | 虚焊            | ❌        |
| 小   | 大   | 飞溅 / 烧穿     | ❌        |
| 大   | 大   | 能耗高 / 不稳定 | ⚠️        |

------

## 十五、为什么点焊要“先压后电、后压冷却”

因为：

- 压力决定 R
- 电流决定 Q
- 冷却阶段需要压力维持焊核成型

------

## 十六、电极端面几何对焊核的决定性影响

> **在点焊中，电极不是“理想点源”，而是一个有尺寸、有形状的实体。**

关键参数：

- 电极端面直径
- 端面形状（平 / 圆弧 / 球面）
- 磨损状态

### 为什么这必须进模型？

```
端面直径 ↑ → 接触面积 ↑ → R ↓ → 单位面积热量 ↓
```

📌 这会直接影响：

- 焊核直径
- 熔深
- 虚焊概率

### 算法视角（你会用得上）

> **融合点不是“几何点”，
>  而是一个由电极端面定义的有效作用区域。**

在简化模型中你可以继续用“点”，
 但文档里必须承认这个**工程近似假设**。

在实际操作中，电极压紧导致母材弹塑性形变，因此融合点应理解为**形变后有效作用区域**，而非理想几何点。

## 十七、异种材料点焊中的“非对称融合”

当两侧材料不同：

- 电阻不同
- 熔点不同
- 导热率不同

结果是：

- 融合区偏向一侧
- 焊核几何中心 ≠ 融合点几何中心

📌 这时：

```
n = normalize(n1 + n2)
```

依然是工程上最稳妥的做法，
 但你要知道它是 **“力学/能量折中法线”**，而不是纯几何法线。

# 终极归档总结（推荐保留）

> **点焊是一个由“几何法线约束 → 压力控制 → 电流热输入 → 融合过程 → 焊核形成”组成的强耦合系统。
> 所谓“垂直焊接”并非目的，而是保证融合能量分布均匀、焊核稳定形成的工程手段。**
