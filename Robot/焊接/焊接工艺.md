# 预备知识

## 一、先把词说清楚：什么是「焊核心」

### 1️⃣ 「焊合」是什么意思？

**焊合（Fusion / Coalescence）**
指的是：

> **焊接过程中，母材与焊材在高温下熔化并相互融合，最终冷却形成连续金属结构的过程**

简单说就是：

> **两块金属“真正变成一块”的那个物理过程**

------

### 2️⃣ 「焊合点」是什么？

在机器人焊接语境里：

> **焊合点 = 焊接过程中电弧 / 激光 / 焊丝作用在工件上的实际能量作用点**

也可以理解为：

- 电弧焊：
  👉 **电弧中心落在工件表面的那个点**
- 激光焊：
  👉 **激光焦点落在材料上的点**
- 点焊：
  👉 **电极压力 + 电流作用的那个接触点**

📌 注意
它**不是几何意义的“两个零件的交点”**
而是：

> **热输入真正发生、金属发生熔合的点**

------

**简短结论先给出（这是你可以直接记住、不会用错的）：**

> **焊合点 ≠ 焊核**
> **焊合点是“位置 / 概念”，焊核是“实体 / 结果”。**

下面我把两者**严格区分 + 建立对应关系**，帮你彻底“落地”。

------

## 一、先给一个最清晰的对照表

| 名称       | 属于什么    | 是不是实体 | 在点焊里指什么              |
| ---------- | ----------- | ---------- | --------------------------- |
| **焊合点** | 位置 / 概念 | ❌ 不是     | 焊接作用发生的中心位置      |
| **焊核**   | 实体 / 结果 | ✅ 是       | 焊合后形成的熔化-凝固金属体 |

一句话：

> **焊合点是“在哪里焊”，焊核是“焊成了什么”。**

------

## 二、从时间维度理解（非常重要）

### 1️⃣ 焊接之前

- 有一个几何位置
- 还没有任何焊接结果

👉 **只能叫“焊合点”**

------

### 2️⃣ 焊接过程中

- 电流通过
- 母材开始熔化
- 冶金结合正在发生

👉 **焊合正在发生，但焊核尚未稳定**

------

### 3️⃣ 焊接完成后

- 熔池冷却凝固
- 内部形成稳定的冶金结合体

👉 **此时才有“焊核”**

------

## 三、为什么工程上“焊合点”和“焊核位置”几乎重合？

因为在点焊中：

- 焊合只发生在极小区域
- 焊核在该区域内形成

所以：

```
焊合点 ≈ 焊核几何中心
```

但注意：

- **一个是点（概念）**
- **一个是体（实体）**

------

## 四、在不同语境下怎么正确用词？

### ✅ 正确用法示例

- “这个焊合点的姿态要对准法线”
- “这个焊点的焊核直径偏小”
- “焊合不足导致焊核不连续”

------

### ❌ 容易被纠正的说法

- ❌“焊合点直径是多少”
- ❌“焊合点太小了”

（这两句里你真正想说的是“焊核”）

------

## 五、把你前面所有知识串成一句完整逻辑

```
焊合点（位置）
   ↓
姿态 / 压力 / 电流控制
   ↓
焊合过程（冶金结合）
   ↓
焊核形成（实体）
```

------

## 六、一句“工程归档级总结”（推荐）

> **焊合点是焊接作用发生的几何位置；
> 焊核是焊合过程成功后在该位置形成的冶金结合实体。
> 二者位置重合但概念不同，不能混用。**

## 二、核心问题：为什么焊接要求机械臂末端点「垂直于焊合点」？

这是**焊接机器人里非常核心、非常“工艺向”的要求**。

我拆成 5 个原因说。

------

## 三、原因 1：保证能量输入是“对称、稳定”的

### 1️⃣ 焊接的本质是「能量输入」

不管哪种焊：

- · → 热
- 激光焊 → 高能光
- TIG / MIG → 电弧 + 熔池

都依赖 **能量集中、可控地进入焊合点**

------

### 2️⃣ 如果末端不垂直，会发生什么？

假设末端（焊枪/激光头）是**倾斜的**：

- 能量分布 **不再以焊合点为中心**
- 一侧热多，一侧热少
- 熔池变形

结果是：

- 焊缝偏移
- 一侧未焊透
- 另一侧过烧

📌 **垂直 = 能量分布轴对称**

------

## 四、原因 2：保证焊缝成形一致（外观 + 强度）

### 1️⃣ 焊缝不是“有没有焊住”就行

焊缝要满足：

- 宽度一致
- 高度一致
- 熔深一致

------

### 2️⃣ 垂直时的熔池形态

当末端 **垂直于焊合点**：

- 熔池呈现 **对称的椭圆或水滴形**
- 金属流动方向均匀
- 冷却收缩对称

------

### 3️⃣ 倾斜时的问题

如果末端歪：

- 熔池被“推向一边”
- 焊道一边高一边低
- 易产生：
  - 咬边
  - 未熔合
  - 气孔

📌 所以工艺上会写：

> 焊枪姿态必须保持垂直于焊合点 ±X°

------

## 五、原因 3：保证焊接熔深（是否焊透）

这个点**现场工程师特别在意**。

------

### 1️⃣ 熔深取决于什么？

- 热量大小
- 热量集中程度
- 热量进入材料的方向

------

### 2️⃣ 垂直的好处

- 热量 **沿材料法线方向进入**
- 最容易达到设计熔深
- 可预测、可重复

------

### 3️⃣ 倾斜的风险

- 热量沿切向扩散
- 表面看起来焊住了
- 实际底部没焊透

👉 **典型“假焊”来源之一**

------

## 六、原因 4：减少焊枪 / 电极 / 焊丝异常磨损

这个是**设备寿命 + 稳定性问题**。

------

### 1️⃣ 末端歪了会怎样？

- 焊丝与熔池接触角度异常
- 电弧偏吹
- 焊枪喷嘴单侧高温

结果：

- 焊丝送丝不稳定
- 焊枪寿命下降
- 飞溅增多

------

### 2️⃣ 垂直的工程意义

- 电弧稳定
- 焊丝对准熔池中心
- 维护周期可控

------

## 七、原因 5：机器人路径规划与工艺参数强绑定

这一点 **你作为做软件 / 算法的人，其实特别关键**。

------

### 1️⃣ 工艺默认假设

焊接工艺参数（电流、电压、速度）**都是在“末端垂直”的前提下标定的**

如果姿态变化：

- 同样的速度
- 同样的电流

👉 焊接效果就不一样

------

### 2️⃣ 为什么机器人系统强调 TCP 姿态？

因为：

> **焊接不是只控制点位（XYZ），而是强依赖姿态（Rx Ry Rz）**

所以现场常见要求：

- TCP Z 轴对准焊合点法线
- 姿态误差 < ±3° 或 ±5°

------

## 八、一句话总结（给你一个“现场能用”的解释）

> **焊接要求机械臂末端垂直于焊合点，是为了让焊接能量以对称、稳定、可预测的方式进入材料，保证焊缝成形、熔深、强度和一致性，同时降低设备异常和工艺波动。**

------

# 什么是虚焊

这个问题**非常关键，而且正好把你前面所有理解（法线、熔深、能量分布、垂直焊接）全部串起来了**。
我用你已经习惯的那套：**现象 → 物理本质 → 几何/能量 → 工程判定**，给你一个**可直接归档的定义**。

------

## 什么是虚焊 / 假焊（Cold Weld / Lack of Fusion）

------

## 一、先给一个“现场不会错”的定义

> **虚焊 / 假焊**：
> 焊接外观上看似已经焊接完成，但**母材没有充分熔化并形成有效冶金结合**，焊缝仅靠表面附着或局部接触维持。

一句话版：

> **“看起来焊住了，实际上没有焊牢。”**

------

## 二、从物理本质解释（最重要）

焊接成立的必要条件是：

> **母材必须熔化，并在液态下与焊缝金属发生冶金结合**

虚焊的本质是：

```
焊缝金属 ≠ 与母材发生充分冶金结合
```

常见表现：

- 焊缝只是“搭”在表面
- 或仅有局部点状结合
- 内部存在未熔合界面

------

## 三、从你已经理解的“熔深”角度看虚焊

### 3.1 正常焊接

```
母材熔化区  ←  熔深足够
──────────
```

### 3.2 虚焊 / 假焊

```
焊缝金属
──────────  ← 表面粘住
母材基本未熔化
```

📌 **熔深不足 = 虚焊的直接根源**

------

## 四、为什么“虚焊”外观往往看不出来？

这是现场最危险的地方。

------

### 4.1 焊缝高度 ≠ 焊接质量

- 焊缝高度大
  ❌ 不代表熔深大
- 焊缝表面连续
  ❌ 不代表内部连续

------

### 4.2 热量被“错误消耗”了

在虚焊中，能量往往被用于：

- 熔化焊丝
- 形成表面焊缝高度
- 表面铺展

而不是：

- 熔化母材
- 建立冶金结合

------

## 五、从能量与姿态模型解释虚焊（与你前面问题强相关）

### 5.1 姿态不合理（非垂直）

当焊接姿态偏离焊合点法线：

```
z_tcp · n < 1
```

结果：

- 能量沿切向扩散
- 熔池“趴在表面”
- 内部熔化不足

👉 **典型虚焊诱因**

------

### 5.2 热输入不足（但表面好看）

```
I · V · t  ↓
```

- 电流不够
- 时间不够
- 速度太快

👉 表面焊住，内部没焊住

------

### 5.3 接触 / 压紧问题（点焊常见）

- 压力不足
- 接触电阻异常
- 电流路径不稳定

👉 焊核小或不连续

------

## 六、虚焊 vs 正常焊接（工程对照）

| 项目     | 正常焊接   | 虚焊 / 假焊  |
| -------- | ---------- | ------------ |
| 外观     | 可好可一般 | 常常“很好看” |
| 熔深     | 足够       | 不足         |
| 冶金结合 | 连续       | 不连续       |
| 强度     | 达标       | 极低         |
| 破坏形式 | 母材断裂   | 焊缝界面剥离 |

------

## 七、为什么“垂直焊接”可以显著降低虚焊概率？

把你前面所有内容串起来：

- 垂直 → 能量对称进入母材
- 熔池向内发展
- 熔深稳定

而虚焊本质是：

> **能量没有进入母材，而是“停留在表面”**

------

## 八、工程上如何判断 / 避免虚焊？

### 8.1 设计 / 工艺阶段

- 明确最小熔深要求
- 姿态强约束（法线对齐）
- 不以外观作为唯一标准

------

### 8.2 检测手段（重点）

- 拉力 / 剪切试验
- 超声 / X 光
- 点焊破坏试验（看焊核）

📌 **没有一种只靠“看”的方法能可靠排除虚焊**

------

## 九、一句话可直接归档的总结（推荐）

> **虚焊 / 假焊是指焊接过程中母材未充分熔化、冶金结合不足，导致焊缝仅在表面形成连接的失效焊接状态；其根本原因是熔深不足，而非焊缝高度不足。**

# 所以核心原因是因为分布不均匀而不是垂直或不垂直

你这个问题**问得非常准**，而且已经抓到本质了。
我直接给你一个**工程级结论**，再把逻辑捋清楚。

------

## 一、先给结论（很重要）

> **是的，核心原因不是“垂直”这个形式本身，而是：
> 👉 能量 / 熔池 / 金属流动是否“分布均匀、可控、可重复”。**

**“垂直”只是目前工程上最简单、最稳定、最容易保证“分布均匀”的姿态。**

------

## 二、为什么大家总说“要垂直”？——这是“工程妥协”

### 1️⃣ 焊接真正关心的不是几何，而是物理场

焊接过程中真正起作用的是：

- 🔥 温度场
- ⚡ 电场 / 能量场
- 🌊 熔池金属流动
- ❄ 冷却收缩场

而不是：

- 点是不是正好 90°
- 轴是不是正交

------

### 2️⃣ 垂直姿态带来的“物理效果”

当末端 **垂直于焊合点法线** 时：

- 能量输入 **轴对称**
- 熔池形状 **左右一致**
- 金属流动 **前后可预测**
- 冷却收缩 **不偏心**

👉 这就是“分布均匀”

------

## 三、那是不是说「不垂直就一定不行」？

**不是。重点来了。**

------

### ✅ 只要你能保证“分布仍然是受控和均匀的”，

**理论上不垂直也可以焊得很好。**

------

## 四、那为什么现场还是强制要求“垂直”？

因为下面这几件事👇

------

### 1️⃣ 工艺参数是“假设你垂直”的

现场的焊接工艺包：

- 电流
- 电压
- 送丝速度
- 行走速度

**全部是在“枪垂直”的前提下实验出来的**

如果你姿态变了：

- 热输入模型变了
- 原参数就不再成立

👉 工艺就失控了

------

### 2️⃣ 机器人不是物理仿真系统

从你做软件的角度看，这点很关键：

- 机器人控制的是
  👉 位姿（XYZ + RPY）
- 它**不实时解算熔池物理模型**

所以工程上要用一个：

> **“简单几何约束”来替代复杂物理控制**

这个约束就是：

> **TCP Z 轴 ≈ 焊合点法线**

------

### 3️⃣ 容错性问题（工业现场非常现实）

假设姿态允许倾斜：

- 焊枪安装误差
- TCP 标定误差
- 工件定位误差
- 轨迹误差

一叠加：

👉 分布立刻严重不均匀

而 **垂直姿态的容错性最大**

------

## 五、换个角度说：垂直 ≠ 目的，而是手段

你可以把逻辑理解成：

```
目的：焊接能量分布均匀、熔池稳定
 ↓
工程上最稳妥的手段：末端姿态垂直
 ↓
软件/工艺统一约定：Z轴对法线
```

------

## 六、一个“软件工程师式”的类比（你一定能懂）

这就像：

- 你写图像算法
- 理论上可以处理任意旋转角度
- 但工程上你会先 **旋正 ROI**
- 再用一套固定参数跑算法

👉 **不是旋正是目的，而是稳定参数空间**

焊接的“垂直”就是这个角色。

------

## 七、什么时候会“刻意不垂直”？

你以后在现场大概率会见到：

### 🔹 摆动焊 / 摇摆焊

- 姿态周期性变化
- 但平均能量仍对称

### 🔹 特殊坡口焊

- V 型 / U 型
- 会设定一个固定倾角
- 同时重算工艺参数

### 🔹 激光焊倾角

- 用于抑制等离子体反射
- 本质还是在**控制能量分布**

------

## 八、一句话压轴总结

> **“垂直”不是焊接的本质，
> “分布是否均匀、是否可控、是否可重复”才是本质。
> 垂直只是目前工业上性价比最高、最稳定的实现方式。**

# 点焊

很好，这三个点**正好是一条完整的工程逻辑链**。
我按你给的顺序来，而且**全部站在“点焊 + 机器人 + CAD”的视角**，不是连续焊那一套。

------

# 一、点焊里：焊合点法线是怎么从 CAD / 曲面算出来的？

> 先一句话总览：
> **焊合点法线 = 焊合点所在接触面的几何法线**
> 点焊本质是 **“两个表面沿法线方向被压合 + 通电”**

------

## 1️⃣ 点焊在 CAD 里到底对应什么几何对象？

在 CAD / 工艺系统中，点焊通常是：

- 两个零件的**接触区域**
- 在接触区域中选取一个**焊点中心**

几何上可以是：

- 面 + 面（最常见）
- 面 + 曲面
- 曲面 + 曲面

📌 **不会是“线”或“体”**

------

## 2️⃣ 法线的来源：面法线 or 曲面法线

### 情况 A：平面（最简单）

如果焊点在一个平面上：

- CAD 面本身就有一个固定法线 **N**
- 焊合点法线 = 平面法线

数学上：

```
平面：Ax + By + Cz + D = 0
法线：n = (A, B, C)
```

------

### 情况 B：曲面（更常见，也更真实）

如果焊点在曲面上（钣金、壳体）：

曲面通常是参数形式：

```
S(u, v) = (x(u,v), y(u,v), z(u,v))
```

#### 曲面法线计算方式：

1. 求两个切向量

```
Su = ∂S/∂u
Sv = ∂S/∂v
```

1. 叉乘得到法线

```
N = Su × Sv
```

1. 单位化

```
n = N / |N|
```

📌 **这个 n 就是焊合点法线**

------

## 3️⃣ 点焊是“两个面”，那到底用谁的法线？

这是点焊里**非常关键的工程问题**。

### 实际做法是：

- **选“电极接触面”的法线**
- 或者 **取两面法线的平均方向**

常见策略：

```
n = normalize(n1 + n2)
```

原因很简单：

- 点焊是 **沿压紧方向施压**
- 电极轴线必须与“等效压紧方向”一致

------

## 4️⃣ 小结一句话

> **点焊的焊合点法线，本质是“焊点所在几何面的局部法线”，
> CAD 给的是几何，工艺拿它当物理压紧方向。**

------

# 二、为什么很多焊接算法本质是「法线跟踪」？

你这个问题已经明显是**算法视角**了。

------

## 1️⃣ 焊接真正要控制的不是点，而是「局部坐标系」

机器人焊接（包括点焊）真正需要的是：

- 一个 **TCP 位姿**
- 而位姿 = 位置 + 姿态

而姿态怎么定义？

👉 **最稳定的方式：用法线定义一个轴**

------

## 2️⃣ 焊接里的“天然坐标系”

在焊合点 P 处，可以天然定义一个坐标系：

- Z 轴：焊合点法线 n
- X 轴：沿焊缝方向 / 任意切向
- Y 轴：叉乘得到

```
Z = n
X = normalize(切向)
Y = Z × X
```

📌 即使是点焊（没有焊缝）：

- Z 轴 **仍然是绝对核心**
- X/Y 只要稳定即可

------

## 3️⃣ 所谓“法线跟踪”在干什么？

> **法线跟踪 = 让 TCP 的某一根轴，持续对齐焊合点法线**

数学上就是：

```
R_tcp * Z_axis ≈ n_surface
```

或者：

```
angle(Z_tcp, n_surface) → 0
```

------

## 4️⃣ 为什么这比“直接算姿态”靠谱？

因为：

- 法线是**几何稳定量**
- 不依赖路径方向
- 不依赖点的前后顺序

这对工业系统意味着：

- 抗噪声
- 抗工件误差
- 抗示教偏差

------

## 5️⃣ 点焊里法线跟踪的意义

点焊虽然是“点”：

- 但每个点都有 **自己的法线**
- 每个点都是一次 **独立的姿态控制**

👉 所以点焊编程其实是：

> **一堆“法线约束位姿”的集合**

------

# 三、把“焊接姿态约束”写成几何 / 数学模型

这部分你应该会觉得**非常顺**。

------

## 1️⃣ 定义变量（工程常用）

- 焊合点：

```
P ∈ ℝ³
```

- 曲面法线：

```
n ∈ ℝ³, |n| = 1
```

- TCP 位姿：

```
T = [R | t]
```

------

## 2️⃣ 核心姿态约束（最重要）

假设 TCP 的 Z 轴是：

```
z_tcp = R * (0,0,1)
```

**姿态约束就是：**

```
z_tcp ⋅ n = 1
```

或者允许误差：

```
z_tcp ⋅ n ≥ cos(θ_max)
```

这就是现场说的：

> “焊枪与焊合点法线夹角 ≤ θ”

------

## 3️⃣ 点焊的完整几何约束模型

### 位置约束

```
t = P + d * n
```

- d：电极伸出量 / 压紧余量

------

### 姿态约束

```
R * Z = n
```

------

### 自由度说明（点焊很关键）

- 绕 Z 轴旋转是 **自由的**
- 点焊不关心 TCP 绕法线转多少度

👉 所以点焊是一个 **欠约束问题**

------

## 4️⃣ 为什么算法上好做？

因为你只需要：

1. 一个点 P
2. 一个法线 n
3. 一个参考方向（防止翻转）

就能唯一构造一个 TCP 姿态

------

## 四、把三点合成一句“工程真话”

> **点焊中：
>
> - CAD 提供几何面 → 算局部法线
> - 焊接算法做的事情，本质是让 TCP 的 Z 轴对齐这个法线
> - 焊接姿态约束，本质是一个“向量对齐 + 位置偏移”的几何问题**

# 点焊中：压力与电流的关系

（为什么“两者必须配合”）

------

## 一、一句话总论（先记这个）

> **点焊不是“电流越大越好”，
> 而是“在合适的压力下，利用接触电阻，把电流转化为稳定的热量”。**

👉 **压力和电流是“强耦合关系”，不是独立旋钮。**

------

## 二、物理与电学本质（关键模型）

### 2.1 点焊的热量从哪来？

点焊热量来源于：

```
Q = I² · R · t
```

- `I`：焊接电流
- `R`：接触电阻（母材接触面 + 电极接触）
- `t`：通电时间

------

### 2.2 压力在这里干了什么？

**压力 ≠ 热源**

压力的作用是：

- 改变接触面积
- 改变接触电阻 `R`
- 稳定电流通路
- 抑制飞溅和间隙

📌 核心关系：

> **压力 ↑ → 接触面积 ↑ → 接触电阻 ↓**

------

## 三、四种典型组合的工程结果（重点）

下面是你问的 **“四象限分析”**。

------

## 情况一：✅ 压力合适 + 电流合适（理想状态）

### 物理状态

- 接触稳定
- 电阻分布合理
- 热量集中在界面

### 结果

- 焊核尺寸合格
- 熔深足够
- 焊点强度高
- 外观稳定

👉 **标准工艺窗口**

------

## 情况二：❌ 压力大 + 电流小

### 发生了什么？

- 压力大 → 接触面积大
- 接触电阻 `R` 显著下降
- 电流又小

代入公式：

```
Q = I² · R · t  →  Q 很小
```

------

### 结果（非常典型）

- 几乎不形成焊核
- 母材未充分熔化
- 表面“压住了”，但没焊住

👉 **典型虚焊 / 假焊**

📌 这是现场**最容易被忽略的一种失败模式**。

------

## 情况三：❌ 压力小 + 电流大

### 发生了什么？

- 压力小 → 接触面积小
- 接触电阻 `R` 很大
- 电流又很大

```
Q = I² · R · t  →  Q 过大
```

------

### 结果（危险组合）

- 局部过热
- 金属剧烈熔化
- 飞溅严重
- 焊核不稳定

常见问题：

- 焊点烧穿
- 表面炸点
- 电极严重磨损

👉 **不是好焊，是“炸焊”**

------

## 情况四：❌ 压力大 + 电流大

### 发生了什么？

- 压力降低了 `R`
- 电流仍然很大

结果取决于平衡点：

------

### 两种可能结果

#### 情况 4A：勉强能焊住

- 焊核形成
- 但：
  - 能耗大
  - 电极寿命短
  - 工艺窗口窄

#### 情况 4B：异常焊接

- 热量转移到表面
- 电极粘连
- 焊点不均匀

👉 **工业上不推荐**

------

## 四、用一张“工程认知表”总结四种情况

| 压力 | 电流 | 结果            | 工程评价 |
| ---- | ---- | --------------- | -------- |
| 合适 | 合适 | 焊核稳定        | ✅ 推荐   |
| 大   | 小   | 虚焊            | ❌        |
| 小   | 大   | 飞溅 / 烧穿     | ❌        |
| 大   | 大   | 能耗高 / 不稳定 | ⚠️        |

------

## 五、为什么点焊“先压紧，再通电”？

这句话你在现场一定听过。

### 原因是：

- 先压 → 稳定接触电阻
- 再通电 → 热量集中在界面
- 通电结束后继续加压 → 焊核冷却成型

📌 **压力是“焊接条件”，不是“焊接能量”**

------

```
姿态（法线对齐）
    ↓
压力方向稳定
    ↓
接触电阻可控
    ↓
I²Rt 转化为有效热量
    ↓
熔深形成焊核
    ↓
避免虚焊 / 炸焊
```

------

## 七、一句话工程级总结（推荐归档）

> **点焊中，压力决定接触电阻与电流通路，电流决定热量规模；
> 两者必须匹配，压力过大配小电流会导致虚焊，压力过小配大电流会导致飞溅和烧穿。**
